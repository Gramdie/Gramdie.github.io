Como usar a Gramdie Games
1. Localizando Jogos

Use a barra de pesquisa no topo para buscar por nomes. Você também pode clicar em "Filtrar" para abrir a busca detalhada, onde pode pesquisar por IDs únicos, datas específicas ou palavras contidas na descrição.

2. O Sistema de Tags (#)

Nossa plataforma usa um sistema de tags para categorias. Digite # na busca seguido do gênero (ex: #Action, #Erotic, #RPG) para listar todos os jogos daquela categoria.

3. Identificando Provedores

Nos cards de jogos, exibimos o ícone de nuvem que indica onde o arquivo está hospedado (Mediafire, Mega, etc). Isso ajuda você a escolher seu servidor de preferência antes de clicar.

4. Modo Escuro/Claro

No canto superior direito, o ícone de sol/lua permite alternar o tema do site. Suas preferências ficam salvas para as próximas visitas.

Segurança em Primeiro Lugar

Nossa plataforma tem tolerância zero para vírus ou malwares. Se você baixou algo e detectou comportamento suspeito, reporte imediatamente para protegermos outros usuários.

Como denunciar:

Caminho: Denunciar > Motivo > Software mal intencionado

Ao enviar o e-mail ou abrir o ticket no Discord, por favor, informe o ID do jogo ou o Título exato para que nossa equipe remova o post imediatamente após a verificação.
