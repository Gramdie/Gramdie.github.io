Sobre o Gramdie Games

Funcionamento

O Gramdie Games é um indexador de links. Nós organizamos jogos e softwares úteis de forma limpa, sem anúncios abusivos.

  • Versão do Site 1.0.4 (Beta)
  • Banco de Dados SheetDB API
  • Atualizações Diárias

Tecnologias

Utilizamos tecnologias modernas para garantir que sua busca seja rápida e o consumo de dados seja baixo.

  • Linguagem HTML5 / JS ES6
  • Ícones FontAwesome 6
  • Estética Inclusive Sans
Segurança e Integridade

Proibição de Softwares Mal Intencionados

Nossa prioridade é a segurança da comunidade. É terminantemente proibido postar vírus ou qualquer script que prejudique o usuário.

Caso encontre algo suspeito, você deve realizar a denúncia imediatamente seguindo este fluxo exato:

Clique em Denunciar > Selecione o Motivo > Software mal intencionado

O desenvolvedor do site não possui vínculo legal com os arquivos postados por terceiros, mas agimos rapidamente para remover conteúdos nocivos denunciados.

FAQ (Dúvidas Comuns)

O link está fora do ar?

Links podem expirar nos servidores de nuvem. Use o botão de suporte para nos avisar e faremos o reupload o mais rápido possível.

Como usar as Tags?

Na barra de pesquisa, use o símbolo # antes da categoria (ex: #RPG). Isso filtra instantaneamente o catálogo por gênero.

Pedido de Jogos
